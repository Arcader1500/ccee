# Practical Machine Learning Syllabus

## Chapter 1: Machine Learning Basics
- Supervised vs Unsupervised Learning
- Feature Engineering: Pre-processing, Scaling, Dimensionality Reduction (PCA, t-SNE)

## Chapter 2: Clustering
- K-Means
- Hierarchical Clustering
- DBSCAN
- Cluster Evaluation

## Chapter 3: Regression Algorithms
- Linear & Polynomial Regression
- Regularization: Ridge, Lasso, Elastic Net
- Metrics: MAE, MSE, RMSE, R2 Score

## Chapter 4: Classification Algorithms
- Support Vector Machines (SVM)
- K-Nearest Neighbors (KNN)
- Decision Trees
- Naive Bayes Classifier

## Chapter 5: Ensemble Methods
- Random Forest
- Gradient Boosting Machines (GBM)
- Model Stacking

## Chapter 6: Time Series Analysis
- Moving Averages, Exponential Smoothing (Holt-Winters)
- Auto-correlation (ACF, PACF)
- ARIMA / ARMA Models

## Chapter 7: Advanced Systems
- **Recommendation Systems**: Collaborative Filtering, Content-Based
- **Anomaly Detection**: Isolation Forest
- Performance Metrics: ROC, AUC, Confusion Matrix, F1 Score
