# NLP & Computer Vision Syllabus

## Chapter 1: Natural Language Processing Basics
- Language in Cognitive Science
- Computational Linguistics: Phonology, Morphology, Syntax, Semantics
- Text Analysis: Tokenization, Stemming, Lemmatization, POS Tagging
- Named Entity Recognition (NER)

## Chapter 2: Parsing and Statistical NLP
- Syntactic & Semantic Parsing
- Information Extraction
- Statistical Models: Markov Models, N-Grams
- Text Classification (SVM, Centroid)

## Chapter 3: Deep Learning for NLP
- Word Embeddings: Word2Vec (Skip-gram, CBOW, GloVe)
- Sequence Models: RNN, LSTM, GRU
- Seq2Seq Models & Attention
- **Transformers**: Architecture, Self-Attention, BERT
- Machine Translation

## Chapter 4: Speech Processing
- Phonetics & Phonology
- Automatic Speech Recognition (ASR)
- Text to Speech (TTS)

## Chapter 5: Computer Vision Basics
- Image Processing Fundamentals
- Edge Detection, Interest Points
- Image Classification

## Chapter 6: Deep Learning for Vision
- **CNN Architectures**: AlexNet, ResNet
- **Object Detection**: YOLO, RCNN, Fast/Faster RCNN, Mask RCNN
- Transfer Learning
- Applications: Face Recognition, Image Generation
